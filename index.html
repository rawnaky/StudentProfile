Project Idea

Overview

a shopping platform that utilises augmented reality to showcase new products in one unified store. This can range from laptops and phones to everyday items like toothbrushes, anything the seller wishes to display a true scale version of their products to replace physical stores saving them significant costs. In the current state of the world, anything that can be done online is being don’t online and so is shopping.

Motivation

The number of online shop spending had skyrocketed in 2020, jumping 41% compared to the average in 2019 and accounting for, although in 2021 it's projected to shrink slightly back to the original numbers for 2021. The main problem with online shopping at the moment is the lack of customer trust due to not being able to touch and feel the products. Sometimes people find themselves checking out a product in person before buying it online. This could be mitigated by using an augmented reality technology to display products so customers wouldn’t need to visit the stores and increasing customer trust in shopping online.

Description

This marketplace would be something similar to Amazon or eBay where sellers would give up a portion of their sales to use the storefront and the AR technology. The store would feature an Ai powered curation system to find products the user is most likely to buy depending on their location, time of day, previous purchases and previously searched products, some spots could be auctioned similar to google AdWord to find the most relevant ad for the highest amount the seller wants to spend to get a new customer. A search bar would be featured in the top section of the page with a camera icon on the right side, pressing which an image selector popup would come up where you can take an image or use an image from camera roll and after selecting an image a similar product or the same product would be shown using an AI image search . Initially, orders would be shipped by the sellers themselves but after the revenue increases, they would be able to choose to stock their products in our warehouses which would be shipped when orders are filled.

Tools and Technologies

For this project, we would need a 3d modelling software such as Cinema 4D or Maya. To show these models Google’s ARcore SDK would need to be implemented into the website which would feature an AR button next to each product page and on the featured page for the big images or web-based app scaled using reactive design. For this website and web app, a cloud service would need to be implemented either google cloud or AWS, google cloud would be a cheaper option and would be better for this project as it has better integration and existing documentation on implementing cloud-based AR solutions and is developed by the same company.

Skills Required

Web app development skills will be required. Developing the backend would require extensive Python skills, and Python would also help use and implement Python-based machine learning and AI vision libraries from Google. This project would also require learning how to implement Google Arcore and other cloud-based solutions and understand their APIs.

Outcome

The goal of this project would help solve the lack of customer confidence when buying products online. The huge surge in online sales due to covid could be maintained by improving customer trust; this can be achieved by implementing AI and AR, and other emerging technologies.
